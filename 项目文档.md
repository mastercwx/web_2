个人博客全站最终开发文档（定稿版）
一、项目整体规范（硬性强制）
1.1 设备适配规则

1. 项目支持 PC 端 + 移动端 双端适配
2. 移动端强制禁止：左右滑动、页面缩放

- 锁定视口、禁止横向溢出、禁止双指缩放
- 所有页面严格自适应、无横向滚动条

3. 全站排版统一，双端 UI 逻辑一致，仅布局自动变化
   1.2 Git 版本管理规范（强制）
1. 主分支：main（仅存放上线稳定代码，禁止开发直接提交 / 合并）
1. 开发分支：dev（测试整合分支）
1. 功能分支：feature/xxx（所有新功能、修改全部在此开发）
1. 合并流程：feature → dev 测试通过 → 审核后合并 main
1. 开发者禁止主动合并主分支
   1.3 全局 UI 核心规范（无 AI 味、极简写实）
1. 主题默认：日间模式（白色基调）
1. 配色体系（全局统一）

- 正文黑灰：#333
- 辅助文字：#777
- 卡片底色：#FAFAFA
- 主背景：#FFFFFF
- 强调按钮色：低饱和橘 #E07848

3. 圆角统一：

- 卡片、弹窗：6px
- 按钮、输入框：4px~5px

4. 禁止：渐变、发光、毛玻璃、夸张动画、立体特效
5. 交互原则：轻交互、无干扰、低动画、高流畅
   1.4 全局性能硬性优化
6. 全站图片浏览器缓存

- 配置 Cache-Control、ETag 缓存策略
- 二次访问优先读取本地缓存，不重复请求服务器

2. 页面流畅度优化

- 图片懒加载
- 长内容分段渲染
- 弹窗、预览层使用硬件加速
- 减少无效重绘、冗余 DOM
  二、前台用户端完整功能规范
  2.1 首页结构

1. 顶部导航（双端适配）

- 左侧留白、中间搜索框、右侧：日夜切换 + 头像
- 搜索框下拉热度 TOP10 列表

2. 文章卡片布局

- PC：双列布局
- 手机：单列铺满、左右自适应边距
- 卡片结构：4:3 封面图 → 头像 / 昵称 / 点赞 / 浏览 → 标题

3. 全局悬浮按钮（右下角竖排固定）

- 主页、消息、个人中心、返回顶部
- 未登录展示登录按钮
  2.2 文章详情页（重点定制功能）

1. 文章图片功能

- 所有图片支持点击全屏预览
- 预览时：底层页面完全锁定禁止滑动、缩放、误触
- 预览层支持：双指缩放、原图下载按钮
- 下载为原图无损保存

2. Markdown 代码块功能

- 所有代码块右上角自带「复制」按钮
- 点击复制后按钮文字切换为「已复制」（极简文字提示，无弹窗）
- 普通正文文本支持原生自由复制
- 代码块样式：浅灰底色、6px 圆角、标准排版、无花哨样式
  2.3 评论系统定制

1. 评论框支持 Emoji 表情插入、发送、正常渲染
2. Emoji 弹窗 从评论框上方弹出

- 绝不遮挡输入框，避免用户输入受阻

3. 评论、回复排版整洁，长内容自动换行、超长回复可折叠
   三、后台管理系统（极简定制版）
   3.1 后台整体 UI 结构（特殊定制）
1. 取消顶部状态栏、顶部导航
1. 全局仅保留底部状态栏（PC / 手机双端统一）
1. 后台布局：左侧菜单 + 右侧内容区

- 手机端：左侧菜单自动折叠为顶部下拉菜单
  3.2 后台基础功能

1. 文章管理
2. 评论管理
3. 用户管理
4. 全站消息推送
5. 数据统计面板（简约线条图表、无花哨配色）
   3.3 新增安全功能（重点）
6. IP 访问统计列表页

- 展示字段：访问 IP、访问次数、最近访问时间、访问路径、操作
- 支持：IP 搜索、日期筛选、一键封禁 IP

2. IP 封禁管理系统

- 可自定义：临时封禁 / 永久封禁
- 可填写封禁原因
- 独立封禁名单页面
- 展示：封禁 IP、封禁时长、封禁原因、操作管理员、解封按钮
- 支持批量解封

3. 封禁逻辑

- 被封禁 IP：前台 + 后台全站拦截，无法访问任何页面
- 所有封禁 / 解封操作写入后台日志永久留痕
  四、全站安全体系（完整落地版）

1. 密码 BCrypt 加盐加密存储，禁止明文
2. 登录防爆破：5 次错误锁定
3. Cookie 开启 HttpOnly + Secure，防 XSS 劫持会话
4. 前后台登录入口隔离，后台独立入口
5. 所有用户输入过滤 XSS、特殊脚本字符
6. 全部接口参数化查询，杜绝 SQL 注入
7. 上传文件校验文件头、重命名、禁止脚本执行
8. 所有编辑 / 删除接口后台校验用户归属权，防越权
9. 评论、点赞、投稿接口限流，防刷接口、灌水
   五、移动端强制规则
10. 禁止页面左右滑动溢出
11. 禁止页面缩放
12. 所有表格、表单、模块自动适配手机宽度
13. 后台表格移动端自动自适应，无横向滚动
14. 所有弹窗、表情面板、图片预览适配手机手势
    六、交互统一标准（全站一致）
15. 所有 hover 效果：仅轻微变色加深，无发光、无缩放、无动画抖动
16. 所有弹窗圆角 6px、细边框、极简风格
17. 按钮统一尺寸、统一配色逻辑
18. 所有提示均为轻量化文字提示，无广告、无多余弹窗
    七、项目最终整体体验目标
19. UI：干净、简约、写实、无 AI 塑料感
20. 流畅：图片缓存 + 懒加载 + 防卡顿渲染
21. 安全：完整防注入、防 XSS、防爆破、IP 封禁体系
22. 体验：图片预览下载、代码一键复制、emoji 评论、无误触
23. 规范：双端完美适配、Git 流程标准化、代码整洁可维护
